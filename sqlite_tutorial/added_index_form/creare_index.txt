что такое индекс?
    индекс - это структура данны, которая повышает скорость поиска строк в таблице данных
зачем нужны индексы?
    индексы ускоряют операции поиска join и сортировки по индексируемым столбцам
какие типы индексов существуют?
    b-tree
    hash
    full-text
    уникальные индесы
что такое уникальный индекс ?
    обеспечивает уникальность значение в столбце, предотвращая дублирование

CREATE INDEX имя_индекса ON таблица (столбец);
CREATE UNIQUE INDEX имя_индекса ON таблица (столбец);
DROP INDEX имя_индекса;

Какие преимущества индексов?
Индексы ускоряют выполнение операторов поиска, сортировки и соединения.

Какие недостатки у индексов?
Индексы занимают дополнительное место и замедляют операции вставки, обновления и удаления.

Как проверить, используется ли индекс?
Используйте команду EXPLAIN перед SQL-запросом
EXPLAIN SELECT * FROM таблица WHERE столбец = значение;
