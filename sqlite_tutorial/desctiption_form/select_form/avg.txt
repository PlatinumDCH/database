форма запроса:
    'SELECT age, AVG(age) FROM Users GROUP BY age'

1. Выборка уникальных значение поля age
    - этот запрос сначала группирует строки таблицы Users по значение в колонке age
2. Выборка среднего значения для каждой группы
    - далее, для каждой уникальной группы значений age, он вычисляет среднее значение возраста
        с помощью агрегатной функции AVG(age)

    SELECT AVG(age) FROM Users;
Нахождение среднего возраста среди все пользователей -> float

    'SELECT CAST(AVG(age) AS INTEGER) FROM Users'
Нахождение среднего возраста среди все пользователей -> int
